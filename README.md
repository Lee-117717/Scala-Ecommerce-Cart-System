# Scala电商购物车系统（期末实践项目）

## 项目简介
基于 **Scala + Slick + MySQL** 开发的简易电商购物车系统，实现商品管理、购物车、订单结算、库存控制等完整流程，具备事务安全、数据一致性、防超卖设计。

## 技术栈
- 语言：Scala
- ORM：Slick（类型安全SQL、事务）
- 数据库：MySQL
- 运行：IDEA + JDK 8

## 核心功能
- ✅ 商品上架、库存管理
- ✅ 添加/修改/删除购物车
- ✅ 订单结算（事务原子性）
- ✅ 库存实时扣减、防超卖
- ✅ 订单查询、支付状态管理

## 项目结构
- src/       Scala源码（实体、DAO、Service、UI）
- docs/      课程设计报告
- screenshots/运行截图

## 项目亮点
- 事务保证：结算全流程原子化，数据一致
- 分层架构：实体/DAO/服务/UI，易维护
- 输入校验：防非法输入、库存校验
- 轻量部署：配置简单、依赖少

## 运行方式
1. MySQL 创建 `ecommerce_cart`
2. IDE 导入项目，配置数据库连接
3. 运行主类，菜单交互
